# Sentiment Analysis Pengguna Platform Twitter Terhadap Program Makan Siang Gratis Pemerintah Indonesia Menggunakan Deep Learning LSTM dan CNN

## Deskripsi Singkat

Repositori ini berisi projek skripsi berjudul **"Sentiment Analysis Pengguna Platform Twitter Terhadap Program Makan Siang Gratis Pemerintah Indonesia Menggunakan Deep Learning LSTM dan CNN"**. Penelitian ini bertujuan untuk menganalisis sentimen masyarakat Indonesia di Twitter mengenai program makan siang gratis yang diusung pemerintah dengan memanfaatkan dua metode deep learning populer: Long Short-Term Memory (LSTM) dan Convolutional Neural Network (CNN).

## Latar Belakang

Program makan siang gratis merupakan salah satu kebijakan pemerintah Indonesia yang menuai banyak reaksi di media sosial, khususnya Twitter. Analisis sentimen dapat membantu memahami opini publik secara luas, memberi insight bagi pemerintah dan masyarakat tentang penerimaan serta tantangan program ini. Melalui pendekatan deep learning menggunakan LSTM dan CNN, penelitian ini berharap dapat menghasilkan model prediksi sentimen yang akurat dan mendalam.

## Metodologi

- **Pengumpulan Data:** Scraping tweet terkait program makan siang gratis.
- **Preprocessing:** Pembersihan teks, normalisasi, tokenisasi, dan stopword removal.
- **Modeling:** Implementasi dan pelatihan model LSTM dan CNN untuk klasifikasi sentimen tweet.
- **Evaluasi:** Analisis hasil akurasi, presisi, recall, dan f1-score kedua model.

## Hasil & Manfaat

Penelitian ini diharapkan memberikan informasi mengenai persepsi publik terhadap program makan siang gratis, serta membandingkan performa LSTM dan CNN dalam tugas sentiment analysis pada data berbahasa Indonesia.
